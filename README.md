# Momentum
노마드코더 바닐라JS 챌린지 17기 졸업작품으로 제출한 모멘텀입니다.
<br/><br/>

## 졸업과제 조건
- 랜덤 배경이미지
- 사용자의 이름을 입력받고 화면에 출력
- 현재 시간 화면에 출력
- 투두리스트 (localStorage 사용)
- 사용자의 위치와 온도, 주간날씨 나타내기 (api사용)
<br /><br />

## 스스로 추가한 조건

- 다크모드 기능 (localStorage 사용)
- 뮤직 플레이어
